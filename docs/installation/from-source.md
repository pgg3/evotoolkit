# Install from Source

For development or to get the latest features from the repository.

---

## Clone the Repository

```bash
git clone https://github.com/pgg3/evotoolkitkit.git
cd evotool
```

---

## Install with uv (Recommended)

```bash
pip install uv
uv sync
```

---

## Install with pip (Editable)

```bash
pip install -e .
```
