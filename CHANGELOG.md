# Changelog

All notable user-facing changes to EvoToolkit are documented here.

## [Unreleased]

## [1.0.0] - 2026-03-19

### Changed

- Clarified public installation guidance and aligned optional extras with the package metadata.
- Replaced placeholder paper citations in public docs with software-version citations.
- Standardized contributor instructions around `uv`, Ruff, MkDocs, and the portable CI test subset.
- Added reviewer-facing documentation describing software deltas from prior domain papers.
- Reframed the public positioning around the reusable framework, reference task families, and tutorial examples.
- Added an explicit reviewed-surface definition and a dedicated coverage configuration for MLOSS reviewer-facing validation.

### Added

- Unified `Method -> Interface -> Task` architecture across EoH, EvoEngineer, and FunSearch.
- Public task families for scientific regression, prompt optimization, adversarial attacks, CUDA engineering, control, and CANN initialization.
- Bilingual MkDocs documentation site and PyPI distribution.
- Cross-platform CI on Linux, macOS, and Windows for Python 3.10-3.12.

### Notes

- `1.0.0` is the reviewed submission release referenced by the MLOSS materials in this repository.
